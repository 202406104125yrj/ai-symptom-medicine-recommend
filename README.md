# AI症状智能用药推荐系统

## 项目简介
本项目是《人工智能导论》课程的期末实践项目，开发了一个基于规则引擎的轻量级AI症状用药推荐系统。用户可以通过自然语言输入身体症状，系统会自动匹配常见疾病并给出对应的非处方药品推荐、用法用量、禁忌说明及温馨提示。

⚠️ **重要声明**：本系统仅为人工智能课程实训项目，不具备专业医疗诊断资质，所有推荐结果仅供学习参考，**绝对不能替代专业医师的诊断与治疗**。如有身体不适，请及时前往正规医院就诊。

## 执行环境
- Python 3.8 及以上版本
- 无任何第三方依赖库，无需额外安装
- 支持 Windows/macOS/Linux 全平台运行

## 核心功能
1. **症状智能识别**：支持用户自然语言输入，自动清洗文本并提取有效症状关键词
2. **疾病匹配推理**：基于结构化知识库，计算症状匹配度并输出最可能的诊断结果
3. **精准用药推荐**：提供药品名称、用法用量、禁忌人群及不良反应等详细信息
4. **特殊人群保护**：自动过滤孕妇、哺乳期妇女、18岁以下等特殊人群的禁忌药品
5. **问诊历史记录**：自动保存所有问诊记录，支持随时查询历史问诊信息
